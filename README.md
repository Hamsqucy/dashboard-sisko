# Website Dashboard School Information System

*Sebuah website dashboard sederhana yang dibuat menggunakan dashboard template Admin LTE dengan bantuan PHP, Javascript, Bootstrap, dan MySQL.*

## Features
- Log In
- Sign In
- Create Data
- Read Data
- Update Data
- Delete Data
- Print Data

## Technologies
- `Visual Studio Code`
- `XAMPP`
- `PHP`
- `Javascript`
- `Bootstrap`
- `MySQL`
